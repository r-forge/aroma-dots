The aroma.affymetrix package

The aroma.affymetrix package is under beta testing and currently available for collaborators. The plan is to go public in summer 2007. Please check back again. In the meantime, here are some peek previews:

  • Copy-number variations displayed by the ChromsomeExplorer (requires Firefox).
  • Spatial representation of probe-level modelling residuals displayed by the ArrayExplorer (requires Firefox).
...and some of its features:
  • Number of arrays: unlimited.
  • System requirements: ~1.5GB RAM, any operating system (easy to install).
  • Chip types: all Affymetrix chip types with a CDF, e.g. SNP, exon, and gene expression.
  • File formats: Works directly with CEL and CDF files.
  • Persistent memory: Final and intermediate results and estimates are stored on file.
  • Robustness: Analysis picks up were last interrupted.
  • Export to/Import from: Bioconductor, CNAG, CNAT, dChip, Affymetrix Power Tools.
  • Pre-processing: Background correction, allelic cross-talk calibration, quantile normalization etc.
  • Probe-level modelling: RMA, MBEI, affine.
  • Post-processing: PCR fragment-length normalization.
  • Copy-number analysis: GLAD.  Possible to combine data from multiple chip types.
  • Genotyping: Interfacing to BRLMM and CRLMM (a bit out of date).
  • Alternative splicing: FIRMA (under development).
  • Dynamic HTML reports: ArrayExplorer, ChromosomeExplorer.
  • Design goals: usability, quality & extendibility.
  • Online documentation: aroma.affymetrix Google Group (will go public with the package)
