The aroma.affymetrix package

by Henrik Bengtsson, 2007-01-17.

Introduction

Currently this package is for share with collaborators. It is very much in a beta version.

Help

Please, always turn to the package help when you get stuck, i.e. 

help.start()

Installation

To install this package do: 

# Loads hbLite() and biocLite()
source("http://www.braju.com/R/hbLite.R")

# Required CRAN packages
install.packages(c("digest", "RColorBrewer"))

# Required Bioconductor packages
biocLite()

# Required Bioconductor packages (developer versions)
biocLite(c("affxparser", "affyPLM"), rver="2.5.0")

# Aroma.affymetrix and other packages of mine
hbLite("aroma.affymetrix")

Additional package

In addition to the above, the following packages is required for some methods of the package to work. The package will given an error noting you about non-installed packages whenever they are required. 

# Fitting the GLAD model for copy-number analysis etc.
biocLite("GLAD")

# Add some support for generating spatial plots
# (EBImage will replace this in the future)
hbLite("R.image")

Developers extensions

# EBImage might not install out of the box on all platforms. 
# See http://www.ebi.ac.uk/~osklyar/EBImage/ for more help!
biocLite("EBImage", rver="2.5.0")

# Fitting the CRLMM model
biocLite("oligo", rver="2.5.0")
